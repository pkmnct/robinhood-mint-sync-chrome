Changelog

Enable or disable the changelog page on extension updates. This changelog shows information about the new features and changes introduced with the update.

Triangle Indicator Fix

In some areas of Mint, there is an expansion panel where the indicator functions counterintuitively. This setting enables a fix so that the indicator for the panel appears expanded when it should.

Debug Mode

Enables logging and disables closing of Robinhood/Mint extension windows for debugging purposes. To see the logs, inspect the background page of the extension.

Trigger Initial Setup

If you need to re-setup the Mint properties, use this button. This will also force a sync. Make sure you are signed in to Robinhood and Mint first.

Trigger Initial Setup

Multiple Accounts

If you would like to enable syncing for multiple accounts, you can enable it here.
Note that this will require you to delete any propeties created before turning on this feature.
Similarly, turning off this feature will require you to delete any properties created while it was on.
Each row should be the name EXACTLY as it appears on top of the Robinhood Accounts page.
Whichever account is logged in at the time of syncing will be updated if found. Only one account can update at a time.
