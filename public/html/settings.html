Changelog

Enable or disable the changelog page on extension updates. This changelog shows information about the new features and changes introduced with the update.

Triangle Indicator Fix

In some areas of Mint, there is an expansion panel where the indicator functions counterintuitively. This setting enables a fix so that the indicator for the panel appears expanded when it should.

Debug Mode

Enables logging and disables closing of Robinhood/Mint extension windows for debugging purposes. To see the logs, enable developer mode in Chrome and inspect the background page of the extension.
